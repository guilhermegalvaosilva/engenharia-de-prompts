## 📚 Atividades Desenvolvidas

### ✅ Atividade 01 – Criação de Prompt Simples

**Objetivo:** Desenvolver um prompt simples, claro e direto, demonstrando compreensão inicial sobre estrutura de instruções para IA.

**Prompt Desenvolvido:**

```
"Explique o que é Engenharia de Prompt como se estivesse ensinando para um estudante
do segundo semestre da faculdade de ADS, utilizando linguagem simples e exemplo prático."
```

**Resultado Obtido:**

> Engenharia de Prompt é a prática de criar instruções claras e bem estruturadas para modelos de Inteligência Artificial, como ChatGPT, Gemini, etc., de forma que produzam respostas mais precisas e úteis.
>
> Pense no prompt como uma receita de bolo: quanto mais detalhada e organizada a receita, melhor será o resultado final.

**Aprendizado Principal:** A inclusão do público-alvo influencia diretamente na qualidade e no nível da resposta gerada, demonstrando a importância do contexto.

## 🎓 Conceitos Principais

### O que é Engenharia de Prompt?

Engenharia de Prompt é a disciplina de criar instruções efetivas para modelos de IA. Envolve:

| Conceito       | Descrição                                                        |
| -------------- | ---------------------------------------------------------------- |
| **Clareza**    | O modelo deve entender exatamente o que você quer                |
| **Contexto**   | Explicar para quem ou para que finalidade a resposta deve servir |
| **Exemplos**   | Mostrar como você quer que a resposta seja estruturada           |
| **Persona**    | Definir o papel ou identidade da IA na interação                 |
| **Restrições** | Especificar limitações de formato, tom ou escopo                 |

## 📖 Principais Aprendizados

✨ **A qualidade do prompt define a qualidade da resposta**

- Prompts bem estruturados geram respostas mais úteis e precisas
- O contexto e a persona são fundamentais para resultados específicos

✨ **Estrutura importa**

- Instruções claras e organizadas produzem melhores resultados
- O formato esperado deve ser explicitamente definido

✨ **Iteração e refinamento**

- A primeira versão do prompt raramente é a melhor
- Refinar baseado nos resultados obtidos melhora significativamente
